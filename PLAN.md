# Nimar PDF convert merge software
# निमाड़ पीडीएफ कंवर्टल एवं मर्ज साॅफ़टवेयर 


## Project Overview
This software is created to convert a document file like docx, xlsx, pptx, doc, xls, ppt, ods html etc to pdf software quickly and easily. The software is in python3 using pyside GUI and other libraries. The project is aimed to be a simple fast and useful tool to facilate conversion of documents on the go. The software will support multiple files to be concatenated into single pdf and also convert into individual pdfs as per user requirement. The software also has a built in pdf reader for quickly checking the pdf.

## Core Features
Following are the salient features of this software :
1. Simple GUI interface that shows option to either upload files. The users can arrange uploaded files with up/down button and can also sort files by name, size or modified date before processing
2. The user can upload multiple finds of any supported extension and software can either concatenate into single pdf or create separate multiple pdf based on options. 
3. The software supports these extensions exclusively : .xls, xlsx, .doc, .docx, .ppt, .pptx, .csv,.txt, .xml,.md, .odt, .ods,.odp, .pdf)
4. After clicking on convert, the software will convert uploaded files, after converting the software will ask for location where to save files and a checkbox to ask whether they want to open created pdf or open containing folder. The save file name in single files should have filename ending with _converted.pdf and merged files should end with _merged.pdf
5. The software should have  complete inbuilt support for devanagari language fonts especially hindi unicode characters 
6. If the user uploads multiple pdf files they are merged together instead of converting.
